    Hugo说明

    Hugo是一个流行的静态网站生成器，用于创建快速、现代化的静态网站。它是一个开源的项目，使用 Go 语言编写，因此生成网站非常快速。Hugo的设计目标是简单易用，同时具有高度的可定制性和灵活性。

    一些 Hugo 的特点包括：

    1. 快速构建速度：由于使用 Go 语言编写，Hugo 在生成静态网站时非常迅速。

    2. 简单易用：Hugo 遵循约定优于配置的理念，使得用户可以快速上手并构建自己的网站。

    3. 主题系统：Hugo拥有丰富的主题库，用户可以选择喜欢的主题并快速应用到自己的网站上。

    4. 支持多种格式：Hugo支持多种内容格式，比如 Markdown、TOML、YAML 等，方便用户书写内容。

    5. 部署简单：由于生成的是静态网站，部署也非常简单，可以方便地托管在各种服务上。

    详细可以参考hugo官网的QuickStart。

    const util = require('util');
const child_process = require('child_process');
const exec = util.promisify(child_process.exec);

function getCreateTimeAsFileName() {
     var d = new Date();
     var year = d.getFullYear();
     var month = d.getMonth()+1;
     var day = d.getDate();
     var hour = d.getHours();
     var minute = d.getMinutes();
     var second = d.getSeconds();
     var time = year+"m"+month+"d"+day+"h"+hour+"m"+minute+"s"+second;
     return time;
}

// execute command function
async function executeCommand() {
     const fileName = getCreateTimeAsFileName()+".md";
     const { stdout, stderr } = await exec('/usr/local/bin/hugo new post/' + (new Date()).getFullYear() + '/' + fileName, 
        {cwd: app.fileManager.vault.adapter.basePath}
     );
     console.log('stdout:', stdout);
     console.log('stderr:', stderr);
     if (stdout) {
         new Notice("New Blog Created["+fileName+"]")
     }else{
         new Notice("New Blog Create Faild. "+stderr)
     }
}

module.exports = async function(context, req) {
    await executeCommand();
}

    静态资源管理之图床

    创建完成文章之后，另外一个困难点就是文章中图片的插入和管理，Obsidian又一款插件Image auto Image plugin，可以结合PIcGO将图片同步到上传到图床，并将外链地址插入到文章中。

    1、插件不需要多余配置

    2、PicGO配置 2.1）Token创建方式

    • 打开 GitHub，点击右上角的头像，然后选择“Settings”。
    • 在左侧导航栏中，点击“Developer settings”。
    • 接着，点击“Personal access tokens”。
    • 点击右上角的“Generate new token”按钮。
    • 为令牌命名（例如“PicGo”），然后勾选“repo”权限。
    • 点击页面底部的“Generate token”按钮。
    • 复制生成的令牌，然后将其粘贴到 PicGo 配置中的“Token”字段。
    • 在PicGo 的“GitHub”配置中填写令牌，然后点击“确认修改”。
